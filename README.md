# 🛒 Grocery List App

A simple and clean React application to manage your grocery shopping list.

## ✨ Features

- Add items to your grocery list
- Mark items as purchased
- Delete items from the list
- Clean and responsive UI

## 🚀 Getting Started

### Prerequisites

- Node.js installed on your machine
- npm or yarn

### Installation

1. Clone the repository:
```bash
   git clone https://github.com/Mshshahabi55/Grocery-List-App.git
```

2. Navigate to the project folder:
```bash
   cd Grocery-List-App
```

3. Install dependencies:
```bash
   npm install
```

4. Run the app:
```bash
   npm start
```

The app will open at `http://localhost:3000`

## 🛠️ Built With

- [React](https://reactjs.org/) - Frontend library
- CSS - Styling
